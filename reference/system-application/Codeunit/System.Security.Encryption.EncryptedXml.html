DecryptDocument

Obsolete

This element will become obsolete from version 24.0. Use DecryptDocument with SecretText data type for EncryptionKey.

Decrypts all EncryptedData elements of the XML document using the specified asymmetric key.

[NonDebuggable]
[Obsolete(Use DecryptDocument with SecretText data type for EncryptionKey.,24.0)]
procedure DecryptDocument(var EncryptedDocument: XmlDocument, EncryptionKey: Text, SignatureAlgorithm: Enum SignatureAlgorithm): Boolean

Parameters

Name Type Description
EncryptedDocument XmlDocument

The XML document to decrypt.
EncryptionKey Text

The asymmetric key to use to decrypt the symmetric keys in the document.

The asymmetric algorithm used to decrypt the symmetric key.

Returns

Type Description
Boolean

Returns true if decryption was successful, otherwise false.
